Página hecha con Angular CLI

COMPONENTES

Esta página esta construida en Angular version 16.2.1, para la primera entregra del curso de ANGULAR de Coderhouse

Se ultizo Angular CLI para generar el proyecto con el comando "ng new <nombre del proyecto>"

La cual se encuentra compuesta de cuatro componentes principales:

  1. Toolbar
  2. Sidebar
  3. Page-wrapper
  4. Footer

Para crearlos se utilizo el comando "ng generate componente <nombre del componente>"
